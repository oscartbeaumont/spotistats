# Spotistats
Spotistats is a tool designed to analyse and backup your music on Spotify!

# Usage

```bash
# npm install netlify-cli -g
git clone https://github.com/oscartbeaumont/spotistats.git
cd spotistats/
./download-assets.sh
netlify dev
```
